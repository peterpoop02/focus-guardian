# FOCUS GUARDIAN

## Overview
Focus Guardian is a smart web application that helps users improve focus using real-time sensor data and productivity tools such as timers and task management.

---

## Features
- Real-time focus tracking
- Pomodoro session counter
- Custom timer
- Daily goal tracker
- Focus score graph
- Todo list

---

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Recharts
- Node.js (API routes)
